# 노션 견적서 뷰어 MVP PRD

## 핵심 정보

**목적**: 프리랜서/솔로 개발자가 노션에서 작성한 견적서를 클라이언트가 웹 URL로 바로 확인하고 PDF로 저장할 수 있게 함
**사용자**: 견적서를 노션으로 관리하는 1인 프리랜서 및 소규모 에이전시, 그리고 견적서를 수령하는 클라이언트

---

## 사용자 여정

```
1. [발행자] 노션 데이터베이스에 견적서 내용 입력
   ↓ 노션 DB에 항목 저장

2. [발행자] 생성된 견적서 고유 URL을 클라이언트에게 전달 (이메일, 메신저 등)
   ↓ URL 공유

3. [클라이언트] 견적서 뷰어 페이지 접근
   ↓ URL 유효성 확인

   유효한 URL → 견적서 뷰어 페이지 → 내용 확인 후 PDF 다운로드
   유효하지 않은 URL → 오류 페이지 표시

4. [클라이언트] 견적서 내용 확인 및 PDF 다운로드
   ↓ 완료
```

## 기능 명세

### 1. MVP 핵심 기능

| ID | 기능명 | 설명 | MVP 필수 이유 | 관련 페이지 |
|----|--------|------|--------------|------------|
| **F001** | 노션 데이터 연동 | Notion API를 통해 지정된 DB에서 견적서 데이터 조회 | 데이터 소스의 핵심, 이 기능 없이는 서비스 불가 | 견적서 뷰어 페이지 |
| **F002** | 견적서 웹 뷰어 | 고유 견적서 ID 기반 URL로 견적서를 웹에서 렌더링 | 클라이언트가 서비스를 사용하는 유일한 진입점 | 견적서 뷰어 페이지 |
| **F003** | PDF 다운로드 | 웹에서 렌더링된 견적서를 PDF 파일로 내보내기 | 클라이언트의 핵심 니즈 (기록 보관, 결재용) | 견적서 뷰어 페이지 |
| **F004** | 견적서 목록 관리 | 발행자가 생성한 견적서 목록 확인 및 URL 복사 | 발행자가 어떤 견적서를 발행했는지 추적하는 핵심 도구 | 견적서 목록 페이지 |
| **F005** | 견적서 미리보기 | 발행자가 클라이언트에게 공유 전 견적서 최종 확인 | 잘못된 내용을 공유하기 전에 검토하는 품질 관리 단계 | 견적서 뷰어 페이지 |

| ID | 기능명 | 설명 | MVP 필수 이유 | 관련 페이지 |
|----|--------|------|--------------|------------|
| **F010** | 발행자 인증 | 패스워드 또는 환경변수 기반 단순 접근 제어 | 발행자 전용 관리 페이지를 무단 접근으로부터 보호 | 로그인 페이지, 견적서 목록 페이지 |

- 클라이언트 서명 기능
- 이메일 자동 발송
- 다국어 지원
- 견적서 템플릿 관리

## 데이터 모델

### 노션 데이터베이스 스키마 (Notion DB 필드)

노션 데이터베이스를 CMS로 활용하므로 별도 DB 없이 Notion 필드 구조를 정의합니다.

| 필드명 | 설명 | Notion 속성 타입 |
|--------|------|----------------|
| id | 견적서 고유 식별자 (URL에 사용) | 페이지 ID (자동) |
| title | 견적서 제목 | Title |
| client_name | 클라이언트 회사명 또는 담당자명 | Text |
| issue_date | 발행일 | Date |
| due_date | 유효기간 | Date |
| status | 견적서 상태 (초안/발행됨) | Select |
| items | 견적 항목 목록 (품목, 수량, 단가) | 페이지 본문 인라인 Table 블록 |
| tax_rate | 부가세율 (기본 10%) | Number |
| notes | 특이사항 및 지불 조건 | Text |
| sender_name | 발행자 이름/회사명 | Text |
| sender_contact | 발행자 연락처 | Text |

### 견적 항목 구조 (Notion 하위 블록)

| 필드명 | 설명 | 타입 |
|--------|------|------|
| item_name | 품목명 | Text |
| description | 품목 상세 설명 | Text |
| quantity | 수량 | Number |
| unit_price | 단가 | Number |
| amount | 금액 (수량 x 단가, 자동 계산) | Number |

## 기술 스택

### 프론트엔드 프레임워크

- **Next.js 15.5.3** (App Router + Turbopack) - React 풀스택 프레임워크, SSR로 Notion API 서버사이드 호출
- **TypeScript 5** - 타입 안전성 보장
- **React 19.1.0** - UI 라이브러리

### 스타일링 & UI

- **TailwindCSS v4** (설정파일 없는 새로운 엔진) - 유틸리티 CSS 프레임워크
- **shadcn/ui** (new-york style) - 고품질 React 컴포넌트 라이브러리
- **Lucide React** - 아이콘 라이브러리

### 폼 & 검증

- **React Hook Form 7.x** - 로그인 폼 상태 관리
- **Zod** - 스키마 검증

### 데이터 소스 & API

- **Notion API** (`@notionhq/client`) - 견적서 데이터 CMS로 활용
  - `databases.query()` - 견적서 목록 조회
  - `pages.retrieve()` - 단일 견적서 조회
  - `blocks.children.list()` - 견적 항목 블록 조회
- **Next.js Route Handlers** - Notion API 키를 서버사이드에서만 사용하도록 보안 처리

### PDF 생성

- **브라우저 `window.print()` + `@media print` CSS** - 1순위 전략
  - 한글 완벽 지원, 브라우저 네이티브 방식으로 가장 안정적
  - 견적서는 단순 테이블 문서이므로 print CSS로 충분한 품질 확보 가능
  - 대안 (print CSS로 부족할 경우): `jsPDF` + 서버사이드 렌더링 조합 검토
  - ~~html2canvas~~ 제외 - 4년간 미업데이트, 한글 폰트 이슈, CSS 지원 불완전

### 인증

- **환경변수 기반 단순 패스워드 인증** - `ADMIN_PASSWORD` 환경변수와 비교
- **서명된 쿠키** - `jose` 라이브러리로 JWT 서명 후 httpOnly 쿠키에 저장
  - 단순 문자열 쿠키는 위조 가능하므로 JWT 서명 방식 사용
  - Next.js Server Action에서 발급, `cookies()` API로 검증

### 배포 & 호스팅

- **Vercel** - Next.js 15 최적화 배포 플랫폼, 환경변수 관리

### 패키지 관리

- **npm** - 의존성 관리

## 핵심 구현 방향

### Notion API 연동 패턴

```
Notion DB 구조:
- 데이터베이스 ID → 환경변수 NOTION_DATABASE_ID
- API 키 → 환경변수 NOTION_API_KEY
- 견적서 페이지 ID → URL 파라미터 [id]

서버 컴포넌트에서 직접 Notion API 호출 (API 키 클라이언트 노출 없음)

견적서 목록 조회:
  databases.query(NOTION_DATABASE_ID) → 페이지 목록 + 속성 반환

견적서 상세 조회 (3단계 호출):
  1. pages.retrieve(pageId)              → 제목, 클라이언트명, 발행일 등 속성
  2. blocks.children.list(pageId)        → 페이지 본문 블록 목록 (table 블록 ID 획득)
  3. blocks.children.list(tableBlockId)  → table_row 블록들 (견적 항목 데이터)

금액 계산: amount = quantity × unit_price (프론트엔드에서 계산, Notion 수식 미사용)
```

### URL 구조

```
/                        → 로그인 페이지 또는 견적서 목록 페이지로 리디렉션
/login                   → 발행자 로그인
/dashboard               → 발행자 견적서 목록 (인증 필요)
/invoice/[notionPageId]  → 클라이언트 견적서 뷰어 (공개)
```

### 환경변수 목록

```
NOTION_API_KEY           노션 통합 API 키
NOTION_DATABASE_ID       견적서 데이터베이스 ID
ADMIN_PASSWORD           발행자 로그인 패스워드
```
